PROJECT NAME : Akinator MEMBERS: 1.Saugat Rasaili 2.Nabin B.k Features of Akinator : 1. Decision Tree – Store questions in a binary tree (Yes → left, No → right). 2.Ask Questions – Print questions in the console, user types “yes” or “no”. 3.Traverse Tree – Based on answer, move to next question. 4.Guess Character – When reaching a leaf node, program guesses the character. 5.Learning Mode – If the guess is wrong, ask the user to teach a new character + distinguishing question. 6.Save Knowledge – Store updated tree in a file so it remembers next time. 7.Replay Option – Ask user if they want to play again. 8.Statistics (Optional) – Show how many correct/incorrect guesses it made.
