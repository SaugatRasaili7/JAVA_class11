PROJECT NAME : Akinator Features of Akinator : Decision Tree – Store questions in a binary tree (Yes → left, No → right). Ask Questions – Print questions in the console, user types “yes” or “no”. Traverse Tree – Based on answer, move to next question. Guess Character – When reaching a leaf node, program guesses the character. Learning Mode – If the guess is wrong, ask the user to teach a new character + distinguishing question. Save Knowledge – Store updated tree in a file so it remembers next time. Replay Option – Ask user if they want to play again. Statistics (Optional) – Show how many correct/incorrect guesses it made.
