PROJECT NAME : Tic Tac Toe (Game) Features of this game: 2-Player Mode – Two players alternate turns. 3×3 Board Display – Show the grid after every move. Valid Move Check – Prevent overwriting occupied cells. Win Detection – Rows, columns, diagonals. Draw Detection – Detect stalemates. Replay Option – Restart after game ends. Player Names – Let players enter custom names. Simple UI Option
